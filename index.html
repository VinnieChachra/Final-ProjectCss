My Projects

Password Protected Door

The Arduino project was a password-protected door with a motor. It allowed access only when the correct password was entered on a keypad. The motor controlled the door lock, granting access to authorized users.

Hangman Game

My C# Hangman game is a word-guessing classic. Players try to uncover a hidden word letter by letter within a limited number of attempts. It's an entertaining and challenging project that utilizes C# programming concepts like loops and arrays to create an engaging word-based game.
